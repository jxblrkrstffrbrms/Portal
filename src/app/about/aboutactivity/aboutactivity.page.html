HOW TO ADD TASK To add a task, simply click the (+) button located below the page.
After clicking, it should be redirected to this page.

After that, you can enter all the details required.You can input the task name, set the priority, pick the date and time, and lastly, choose a category.
To save, click the "ADD TASK" button. HOW TO UPDATE TASK To update a task, simply click the update button on the task that you want to update.
After clicking, it should be redirected to this page.

After that, you can enter all the details required.You can update the task name, set the priority, pick the date and time, and lastly, choose a category.
To save, click the "UPDATE TASK" button. HOW TO DELETE TASK To delete a task, simply click the delete button on the task that you want to delete.
